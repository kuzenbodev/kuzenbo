# Support

Thanks for using Kuzenbo.

## Where To Get Help

- Bug reports: use the Bug Report issue form in GitHub Issues.
- Feature requests: use the Feature Request issue form in GitHub Issues.
- Documentation improvements: use the Docs Improvement issue form.

## Security Reports

Do not open public issues for vulnerabilities.

Please follow our security reporting process in `SECURITY.md`.

## Response Expectations

- We review community issues as quickly as possible.
- Security reports receive an initial response target of 3 business days.
