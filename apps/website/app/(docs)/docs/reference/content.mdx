## What This Section Covers

Use this section when you need quick answers about package scope, upgrades, compatibility, and support.

## Public Package Scope

These packages are part of the public Kuzenbo surface:

- `@kuzenbo/core`
- `@kuzenbo/hooks`
- `@kuzenbo/charts`
- `@kuzenbo/notifications`
- `@kuzenbo/date`
- `@kuzenbo/ai`
- `@kuzenbo/datatable`
- `@kuzenbo/styles`
- `@kuzenbo/theme`

If a package is not documented in this site, treat it as internal or not yet stable for public usage.
