title: "useActiveTooltipLabel"
description: "Read the active tooltip label from Recharts interaction state to drive custom headers, badges, or synchronized UI."
---

## Overview

`useActiveTooltipLabel` exposes the current tooltip label (for example an x-axis category like `"Feb"`) from Recharts state.

## What It Reads and Writes

- Reads: active tooltip label from chart interaction state.
- Writes: nothing. This hook is read-only.

## When to Use

- Show a small "currently hovered" label above or below a chart.
- Sync nearby UI to the same category/value currently highlighted by the tooltip.

## Installation

```npm
npm i @kuzenbo/charts recharts
```

## Import

```tsx
import { useActiveTooltipLabel } from "@kuzenbo/charts/ui/chart";
```

## Practical Example

```tsx
"use client";

import {
  Chart,
  type ChartConfig,
  useActiveTooltipLabel,
} from "@kuzenbo/charts/ui/chart";
import { Bar, BarChart, XAxis, YAxis } from "recharts";

const data = [
  { month: "Jan", visits: 120 },
  { month: "Feb", visits: 148 },
  { month: "Mar", visits: 170 },
];

const config: ChartConfig = {
  visits: { label: "Visits", color: "var(--color-chart-2)" },
};

const ActiveLabelBadge = () => {
  const activeLabel = useActiveTooltipLabel();

  return (
    <p className="text-sm text-foreground">
      {activeLabel === undefined
        ? "No active point"
        : `Active label: ${String(activeLabel)}`}
    </p>
  );
};

export function ActiveTooltipLabelExample() {
  return (
    <Chart.Root className="h-72 w-full" config={config}>
      <BarChart data={data}>
        <XAxis dataKey="month" />
        <YAxis />
        <Bar dataKey="visits" fill="var(--color-visits)" radius={6} />
        <Chart.Tooltip content={<Chart.TooltipContent />} />
      </BarChart>
      <ActiveLabelBadge />
    </Chart.Root>
  );
}
```

## Return Contract

- Type: `string | number | undefined`
- `undefined` means no active interaction or missing chart context.
- For category charts, labels are usually strings. Numeric domains may produce numbers.

## Caveats

- Label values come from tooltip state, so they depend on chart type and axis configuration.
- If you need the exact hovered datum payload, pair this hook with `useActiveTooltipDataPoints`.

## SSR and RSC Notes

- Use in Client Components only.
- Do not call from React Server Components.
