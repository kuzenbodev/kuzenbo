## Overview

`Form` is the canonical form root primitive in `@kuzenbo/core`.
It wraps Base UI form behavior and standardizes layout, validation wiring, and submit handling.

## Installation

```npm
npm i @kuzenbo/core
```

## Import

```tsx
import { Form } from "@kuzenbo/core/ui/form";
```

## Basic Usage

```tsx
import { Button } from "@kuzenbo/core/ui/button";
import { Field } from "@kuzenbo/core/ui/field";
import { Form } from "@kuzenbo/core/ui/form";
import { Input } from "@kuzenbo/core/ui/input";

export function AccountForm() {
  return (
    <Form noValidate>
      <Field>
        <Field.Label htmlFor="email">Email</Field.Label>
        <Field.Control render={<Input id="email" type="email" />} />
      </Field>
      <Button type="submit">Save</Button>
    </Form>
  );
}
```

## API Notes

- `Form`: main root export.
- `FormRoot`: explicit root primitive for composition-heavy wrappers.
- Base UI form props are supported (`onSubmit`, `onInvalid`, validation mode controls).
