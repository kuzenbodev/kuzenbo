## Purpose

`@kuzenbo/date` provides Mantine-equivalent date primitives with Kuzenbo conventions:

- `selectionMode` (`single | multiple | range`) across picker surfaces.
- `withWeekNumbers` support for ISO-style week columns.
- Input parity controls (`closeOnChange`, `clearable`, and free typing in `DateInput`).
- `date-fns`-powered adapter layer with timezone/locale support.
- RTL-aware navigation through `DatesProvider`.
- Calendar + picker + input + time primitives from one package root.

## Installation

```bash
bun add @kuzenbo/date @kuzenbo/core @kuzenbo/theme
```

## Basic Calendar

```tsx
import { Calendar } from "@kuzenbo/date";

export function BillingMonthPreview() {
  return <Calendar className="rounded-lg border" selectionMode="single" />;
}
```

## Range Picker

```tsx
import { useState } from "react";
import { DatePicker } from "@kuzenbo/date";

export function RangePicker() {
  const [value, setValue] = useState<[Date | null, Date | null]>([null, null]);

  return (
    <DatePicker
      selectionMode="range"
      value={value}
      onChange={(nextValue) => {
        setValue(nextValue as [Date | null, Date | null]);
      }}
    />
  );
}
```

## Date Inputs

```tsx
import { DateInput, DatePickerInput } from "@kuzenbo/date";

export function DateFieldExamples() {
  return (
    <div className="space-y-3">
      <DateInput
        clearable
        defaultValue={new Date(2026, 1, 14)}
        placeholder="Type or pick (YYYY-MM-DD)"
      />
      <DatePickerInput
        closeOnChange={false}
        name="releaseWindow"
        placeholder="Keep dropdown open for range picking"
        selectionMode="range"
      />
    </div>
  );
}
```

## Week Numbers

```tsx
import { Calendar } from "@kuzenbo/date";

export function WeeklyPlanningCalendar() {
  return (
    <Calendar
      defaultMonth={new Date(2026, 1, 1)}
      selectionMode="single"
      withWeekNumbers
    />
  );
}
```

## Date + Time

```tsx
import { DateTimePicker, TimePicker, TimeValue } from "@kuzenbo/date";

export function Scheduler() {
  return (
    <div className="space-y-3">
      <DateTimePicker name="appointment" withSeconds />
      <TimePicker interval={15} withSeconds />
      <TimeValue value="14:30:00" withSeconds />
    </div>
  );
}
```

## Locale + RTL + Timezone

```tsx
import { DatesProvider, Calendar } from "@kuzenbo/date";

export function LocalizedCalendar() {
  return (
    <DatesProvider
      direction="rtl"
      firstDayOfWeek={6}
      locale="ar-SA"
      timeZone="Asia/Riyadh"
    >
      <Calendar selectionMode="single" />
    </DatesProvider>
  );
}
```

## Notes

- Use `selectionMode` (`single`, `multiple`, `range`) as the only supported mode prop on calendar and picker surfaces.
- Configure locale, direction, and timezone through `DatesProvider` root props, including runtime switches.
- `className` is supported for Tailwind composition; style escape hatches are intentionally removed.
