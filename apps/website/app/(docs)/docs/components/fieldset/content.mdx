## Overview

`Fieldset` groups related form controls under a shared legend.
Use it for option clusters like notification channels, billing preferences, or grouped security settings.

## Installation

```npm
npm i @kuzenbo/core
```

## Import

```tsx
import { Fieldset } from "@kuzenbo/core/ui/fieldset";
import { Field } from "@kuzenbo/core/ui/field";
import { Input } from "@kuzenbo/core/ui/input";
```

## Basic Usage

```tsx
import { Fieldset } from "@kuzenbo/core/ui/fieldset";
import { Field } from "@kuzenbo/core/ui/field";
import { Input } from "@kuzenbo/core/ui/input";

export function ContactPreferencesFieldset() {
  return (
    <Fieldset>
      <Fieldset.Legend>Contact preferences</Fieldset.Legend>

      <Field>
        <Field.Label htmlFor="phone">Phone</Field.Label>
        <Field.Control render={<Input id="phone" type="tel" />} />
      </Field>

      <Field>
        <Field.Label htmlFor="email">Email</Field.Label>
        <Field.Control render={<Input id="email" type="email" />} />
      </Field>
    </Fieldset>
  );
}
```

## API Notes

- `Fieldset`: convenience root namespace.
- `Fieldset.Root`: explicit root primitive.
- `Fieldset.Legend`: legend primitive for group labeling.
