## Legacy alias

`DropdownMenu` was renamed to `Menu` in `@kuzenbo/core`.

Sizing behavior is identical to `Menu`:

- Canonical tokens: `xs | sm | md | lg | xl`
- Default: `md`
- Cascade: `child size ?? content size ?? root size ?? "md"` for overlay surfaces.

Use the canonical page for current API names and subprimitives:

- [Menu](/docs/components/menu)
