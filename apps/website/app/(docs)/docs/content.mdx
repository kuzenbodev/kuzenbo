## Welcome

Kuzenbo is a React UI library built on Base UI and Tailwind, with design tokens, dark mode out of the box, and modular packages. Install only what you need and build fast.
In almost all apps, install `@kuzenbo/theme` alongside the Kuzenbo UI packages you use.

## What's included

Kuzenbo ships as separate packages so you can keep bundles lean:

- **core** — buttons, inputs, dialogs, menus, forms, layout
- **charts** — prebuilt charts + Recharts primitives
- **date** — calendar and date pickers
- **notifications** — toast system
- **ai** — AI widget and session hooks
- **datatable** — table primitives with TanStack
- **theme** — tokens and light/dark runtime
- **styles** — optional global baseline CSS

## Quick setup

Add theme CSS and the provider to your root. For Next.js App Router:

```tsx
import type { ReactNode } from "react";

import "@kuzenbo/theme/prebuilt/kuzenbo.css";
import "@kuzenbo/styles/recommended.css";
import { ThemeBootstrapScript, ThemeProvider } from "@kuzenbo/theme";

export default function RootLayout({ children }: { children: ReactNode }) {
  return (
    <html lang="en" suppressHydrationWarning>
      <head>
        <ThemeBootstrapScript />
      </head>
      <body>
        <ThemeProvider>{children}</ThemeProvider>
      </body>
    </html>
  );
}
```
